# Mrmans0n Tap

## How do I install these formulae?

`brew install mrmans0n/tap/<formula>`

Or `brew tap mrmans0n/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "mrmans0n/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
